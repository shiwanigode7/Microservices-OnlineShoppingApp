# Microservices-OnlineShoppingApp
Online shopping app using micro-service architecture

Service1 - Product Service - Create and View Products, acts as Product Catalog
Service2 - Order Service - Can Order Products
Service3 - Inventory Service - Can check if product is in stock or not
Service4 - Notification Service - Can send notifications, after order is placed
